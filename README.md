# Personal Portfolio Website

This is a personal portfolio website built using React and Vite. It showcases my projects, skills, and experience in web development.

## Features

- Responsive and modern user interface
- Project showcase section with detailed descriptions
- Skills and technologies overview
- About Me section highlighting experience and background
- Contact section for easy communication
- Fast performance powered by Vite
- Smooth navigation and user-friendly design
- Reusable and modular React components
- Easy customization for personal branding and content updates

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3
- Git & GitHub
